# Student Planner

Student planner desktop app built with Python and `tkinter`.

## Features

- Dashboard with pending tasks, completion rate, habit rate, next exam, and a daily quote
- To-Do List with priorities, completion toggle, delete action, and progress bar
- Exam Schedule with subject, date, time, hall, and live countdown
- Weekly Timetable with color-coded subject cards
- Study Timer with Pomodoro ring animation and work/break settings
- Habit Tracker with a weekly tap-to-mark board
- Notes page with local autosave

## Run

```bash
python app.py
```

## Data Storage

The app saves your data automatically to:

```text
data/student_dashboard_data.json
```

## Requirements

- Python 3.11+
- No external packages required
